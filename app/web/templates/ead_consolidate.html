What does consolidation do?
  1. Canonical Rename — Each file's raw headers are renamed to standard ECL field names (e.g., DrsPOS → outstanding_principal).
  2. Stack — All rows from all files are combined into one dataset. Missing columns are filled with nulls.
  3. Source Tracking — A _source_file column is added so each row can be traced back to its original file.
  4. Excel includes a Summary sheet listing all source files, total rows, and generation timestamp.
